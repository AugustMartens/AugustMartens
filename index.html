Projects

Here are a few of my projects, both personal and university ones.

Soundcafe

In 2012 (Age 15), I started a music blog with two friends to share the playlists we made. At the time, I was already interested in web development, and this provided the opportunity to make a nice website with Grooveshark (RIP) integration.

EE 4: Electronic game

The first big electronics project I did was during my fourth engineering experience course. In these courses, we learn to apply theory to practice. In this case, we made an electronic game based on the existing game "Super Hexagon". We programmed in embedded-c a PIC18F-family microcontroller by Microchip. The player can move an arrow with a stepper-motor to avoid the incoming lights and gain points.

Software Engeneering: Bike sharing platform

During my course "Software Engineering", we could develop a software project of our choice. I opted for a bike-sharing platform which I executed by generating an android application and a smart lock. The android application could scan a QR code on the lock to unlock the bike. Then the app monitored the distance traveled and charged the user accordingly through Paypal. I made use of a lot of design patterns and good coding practices. Our project was evaluated on performance metrics like CPU and memory utilization.

For Fun: An electric skateboard

When my bike got stolen again in Leuven, I wanted a new, more efficient way of transporting myself. It was also around the time Casey Neistat got famous with his boosted board skateboard. So my friend and I decided to make our own electric skateboard by choosing off the shelve component meant for model airplanes. We soldered a battery pack with a BMS-Management system and modified an ESC controller for our brushless DC-Motor. IT still works great! Right now, I am also the manager of the Electric Skateboarders Belgium Facebook group!

Drone Air quality Monitoring

As my bachelor project, we made an autonomous Air quality monitoring system that can be mounted to a drone to make live measurements of airborne pollutants. This project was in cooperation with the city of Leuven. The project combined a wide range of sensors, all using different communication standards (I2C, UART, SPI) and sending the data over a LORA-WAN network to our database, which updates an online dashboard for the city's environment council.

Thesis: a smart speaker for the belgian market

Finally, I am currently working on my masters' thesis regarding the implementation of a Natural Language Processing System for the Dutch/Flemish Language. The smart assistant speaker recognizes wake-words and listens to the user's voice input. It processes the user's requests using speech to text and machine learning tools to gather data from external APIs and answer in a fluent sentence + visual representation.
